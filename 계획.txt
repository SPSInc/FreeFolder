월 : 스크립드 완성
화~수: Tag Read/Write
     bank 00 : reserved :: killPW(16bit), accessPW(16bit), 나머지는 쓰레기값
     bank 01 : EPC :: crc(16bit), pc(16bit), EPC(길이만큼;Tag마다 제한된 길이 있음)
     bank 10 : TID :: Tag정보 (규격 모델 회사… 등등) / write 할수 없음
     bank 11 : UserMamory
    * Memory Read 기능은 잘됨
    * 실험한 Asterisk Tag(ImpinJ/Monza5)는 영구락이 걸려있어서 Write가 안됨(팀장님과 확인)
    * 다른 Tag(ImpinJ/Monza4)로 Write 확인 
    * 각 bank 마다 Start Address는 0부터 시작함
    * AretePOP reader, Phychips 프로그램과 Dongle reader 동시 확인함
